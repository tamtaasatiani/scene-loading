scene-loading is a small Unity project focused on asynchronous scene loading, loading screens, and scene management.

# Key features
* Loading screens implemented with asynchronous programming via UniTask
* "SceneData" scriptableobjects for scene verification

# Packages
Unity packages used by the project
* [UniTask](https://github.com/Cysharp/UniTask) by Cysharp Inc | Loading screen, scene loading
* [Unity Terrain - URP Demo Scene](https://assetstore.unity.com/packages/3d/environments/unity-terrain-urp-demo-scene-213197?srsltid=AfmBOoqRra5t4BSYUryzYMekMHMuRP_SSLtpE9_Su9tY78sLcIv6t9Ru) by Unity Technologies Inc | "Heavy" Demo scene
* Probuilder by Unity Technologies Inc | "Light" Demo scene
* Screen Recorder by Unity Technologies Inc | Screenshot recording
